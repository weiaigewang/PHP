# MessageBoard

------

本项目是一个php的简单的demo，**留言板** 自己做的一个小例子 —— 里面没有用到PHP框架，MySQL数据库做支持：

> * 创建MySQL数据库名为studentsys
> * 搭建PHP运行环境，也可以直接用phpstudy
> * 开发Apache或者phpstudy,运行项目
> * 通过浏览器访问，ip+MessageBoard/login.php
